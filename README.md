# Vuekan
VueJS Kanban Board

Only 3 fields - To do, Doing, Done

Items can be dragged using the Dragula library. 
Items can be scrolled independent of the other columns.
See a total count of items in each column.

## Keyboard Shortcuts

Alt/Cmd-N - New Tasks
